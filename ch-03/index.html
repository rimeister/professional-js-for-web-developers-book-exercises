Chapter 3: Language Basics

Data Types

The Undefined Type

Undefined data types are variables which have been declared, but not initialized.

In this example, var foo; has been declared, but is uninitialized.

Code

<script type="text/javascript">
var foo;
</script>
<a href="#" onClick="event.preventDefault();alert(typeof foo);">Click here to see what data type foo is.</a>

The Null Type

A null data type is an object with no value.

In this example, var bar has been set to null. Alerting typeof bar will return object.

Code

<script type="text/javascript">
var bar = null;
</script>

The Boolean Type
